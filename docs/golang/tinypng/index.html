    批量图片压缩go实现

    2021-08-03
    go, tinyPng

    基于golang和tinypng的图片批量压缩工具实现

    准备工作 #

    1. 首先通过 tinyPNG获得一个api_key用于压缩图片api的验证，输入名称+邮箱，然后打开收到的邮件里的链接即可获得，一个key每月免费500次，即一个月可以压缩500张图片，500张对我个人使用够用了。
    2. 然后创建一个码云的公共仓库用来存储压缩后的图片，gitee账号设置里面生成一个access_token用于提交图片。 【PS：如果不用作图床，只需要把压缩后的图片存储到本地的话，可以不需要此步骤】
